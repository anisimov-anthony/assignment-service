# Assignment Service

Микросервис для автоматического назначения ревьюверов на Pull Request'ы.

## Описание

Сервис автоматически назначает ревьюверов на PR из команды автора, позволяет выполнять переназначение ревьюверов и получать список PR'ов, назначенных конкретному пользователю, а также управлять командами и активностью пользователей.

## Быстрый старт

### Запуск через Docker Compose

```bash
docker-compose up
```

Сервис будет доступен на `http://localhost:8080`

**Примечание:** Если у вас уже запущен локальный MongoDB на порту 27017, Docker Compose автоматически использует порт 27018 для внешнего доступа. Внутри Docker сети контейнеры общаются по стандартному порту 27017.
